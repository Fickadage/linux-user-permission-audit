# Cybersecurity Login Monitoring (SQL Project)

Author: Gbenga Adeyinka. 

## Purpose
This project simulates a security monitoring database that tracks user 
login attempts and identifies suspicious activity.

## Features
- User account tracking
- Login attempt logging
- Failed login detection
- Suspicious IP identification
- Security event analysis

## Security Concepts Demonstrated
- Authentication monitoring
- Brute force detection
- Log analysis
- SQL joins for event correlation

## Example Detection Queries
- Multiple failed login attempts
- Suspicious IP addresses
- Login success vs failure patterns

## Technologies Used
- SQL
- MySQL
- Relational database design
- Security event analysis
